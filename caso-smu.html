SMU - Retail

Proyecto: Conversión a SAP S/4HANA y migración desde la nube privada de Entel hacia la nube pública de Huawei.

Descripción del Proyecto

Antes del proyecto, SMU operaba con un ecosistema SAP compuesto por: SAP ERP 6, SAP Solution Manager, SAP NetWeaver 7.5 PO (Process Orchestration), SAP GRC (Governance, Risk, and Compliance), SAP WMS (Warehouse Management System), SAP WebDispatcher y SAP Router. El proyecto consistió en la conversión de SAP ERP 6 a S/4HANA 2022 utilizando un enfoque de "System Move", que permitió mantener el sistema fuente operativo durante la conversión y migración. Se migraron datos a Huawei Cloud con herramientas de terceros para la selección de datos específicos, optimizando el tiempo de inactividad. Se consolidaron aproximadamente 34 TB de datos en la nube de Huawei, utilizando una copia del sistema PRD como base para los entornos DEV y QAS.

Duración

El proyecto tuvo una duración de 11 meses calendario, abarcando análisis, conversión, migración, pruebas y estabilización post-Go Live.

Inconvenientes

Se encontraron inconvenientes en la remediación de datos transaccionales y financieros, la eliminación de add-ons de terceros no soportados, las limitaciones en el ancho de banda entre el sitio principal y la nube de Huawei, la necesidad de conservar el direccionamiento IP existente, las capacidades insuficientes en la nube de Huawei que generaron interrupciones en la operación, y los problemas con interfaces no documentadas que no fueron probadas adecuadamente.

Factores de Éxito

  • Uso de herramientas de migración selectiva.
  • Colaboración con expertos externos.
  • Enfoque en la modernización tecnológica.

Resultados

  • Modernización a S/4HANA 2022.
  • Consolidación de infraestructura en Huawei Cloud.
  • Optimización del rendimiento del sistema.
